# Solidity Basics & HealthTech Exploration

## 🩺 About Me
I am a **Remote Doctor from Nigeria** currently bridging the gap between clinical medicine and Web3 technology. My goal is to leverage blockchain to improve maternal health outcomes and decentralize medical records.

## 🚀 Current Project: Simple Storage
This repository contains my foundational work in **Solidity**, focusing on:
* **Structs & Arrays:** Managing complex data types.
* **Mappings:** Implementing efficient "Key-Value" lookups (essential for patient ID systems).
* **Gas Optimization:** Understanding how to write cost-effective code on the Ethereum network.

## 📈 The Vision
This is the first step in a larger journey to develop a **Digital Partograph** on the blockchain—ensuring that critical labor monitoring data is immutable, transparent, and accessible across borders.

## 🛠 Tech Stack
* **Language:** Solidity
* **Environment:** Remix IDE
* **Version Control:** Git/GitHub
